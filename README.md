# Web Data Extractor – Top USA Companies

## 📌 Project Overview

This project scrapes data from Wikipedia to extract a list of the largest companies in the United States by revenue.

It collects structured data and exports it into a CSV file for analysis.

---

## ⚙️ Features

* Scrapes real-world data from a website
* Extracts table data (company, revenue, rank, etc.)
* Cleans and structures the data using pandas
* Exports data to CSV format

---

## 🛠️ Tech Stack

* Python
* requests
* BeautifulSoup (bs4)
* pandas

---

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the script:

```
python main.py
```

---

## 📂 Output

The extracted data will be saved in:

```
data/Top_USA_Companies.csv
```

---

This project demonstrates:

* Web scraping
* Data extraction

* Automation using Python
